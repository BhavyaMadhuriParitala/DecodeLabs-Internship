# AI ChatBot 🤖

## Description
Built a rule-based AI chatbot in Python using dictionaries and the `re` library. Implemented input sanitization, personalized greetings, efficient O(1) response lookup, conversation handling, fallback responses, and user-friendly command interaction.

## Features
- Personalized greetings
- Input sanitization using Regular Expressions
- Fast dictionary-based response retrieval
- Greeting and farewell handling
- Fallback responses for unknown inputs

## Technologies Used
- Python
- Regular Expressions (`re`)
- Dictionaries (Hash Maps)

```bash
python chatbot.py
